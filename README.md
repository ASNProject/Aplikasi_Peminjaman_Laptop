# Aplikasi_Peminjaman_Laptop

## Packages
Install Packages
```
pip install pillow
pip install tk
pip install future
```
## Run & Debug
Clone this project
```
git clone https://github.com/ASNProject/Aplikasi_Peminjaman_Laptop.git
```
Run program
```
python main.py
```
